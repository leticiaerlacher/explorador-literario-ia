# 🧪 Explorador Literário e Científico Inteligente

Sistema de recomendação de livros desenvolvido como projeto de **Iniciação Científica** no Centro Universitário Multivix. A aplicação utiliza Inteligência Artificial para conectar literatura e temas científicos de forma lúdica e acessível.

## 🤖 Arquitetura de IA e NLP
O motor de recomendação deste projeto baseia-se em conceitos de **Processamento de Linguagem Natural (NLP)**:

- **Vetorização TF-IDF:** O algoritmo analisa a importância de cada palavra nos resumos e títulos, transformando textos em vetores numéricos.
- **Similaridade de Cosseno:** Utilizada para calcular a distância matemática entre a busca do usuário e o acervo, garantindo recomendações semanticamente próximas.
- **Acessibilidade Digital:** Integração com **gTTS (Google Text-to-Speech)** para narração automática dos resumos.

## 🛠️ Tecnologias Utilizadas
- **Linguagem:** Python (Streamlit, Pandas, Scikit-Learn)
- **Engenharia de Dados:** TF-IDF e Similaridade Vetorial
